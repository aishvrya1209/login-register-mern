"I built a full-stack MERN authentication system where users can register, log in, and access protected profile routes using JWT authentication. Passwords are securely hashed with bcrypt before being stored in MongoDB. The backend uses Express middleware to verify JWTs and protect APIs, while the React frontend manages authentication state using localStorage along with ProtectedRoute and PublicRoute components to control access based on login status."
